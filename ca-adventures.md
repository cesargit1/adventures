These sample records are drafted from the `public.adventures` schema in `database/schema.sql` and field usage in `src/components/adventure/AdventureForm.tsx`.

- Scope: one drafted adventure per California map location currently pinned.
- Status: all records use `open`.
- Year: all `start_at` dates are in 2027.
- Season: included as a **draft helper field** (not a DB column) to support planning.
- Host assignment for seeding: alternates between profile IDs for usernames `ceez` and `danny`.
